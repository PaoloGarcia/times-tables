## Notes:
Generates times tables as text files

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Base for the times table to be created      [number] [required]
  -s, --size     size of the times table                  [number] [default: 10]
  -l, --list     Show table created                   [boolean] [default: false]
```
